## üìã Common Tasks

### For Administrators

#### Add New Homeowner
1. Login to admin panel
2. Click "Homeowners" ‚Üí "Add New"
3. Fill in required information:
   - Full name
   - Email address
   - Phone number
   - Lot/Block number
4. Upload photo (optional)
5. Click "Generate QR Code"
6. System sends email to homeowner

#### Revoke QR Code
1. Search for homeowner
2. Click "QR Code Management"
3. Click "Revoke Current QR Code"
4. Click "Generate New QR Code"
5. New QR code sent automatically

#### Generate Reports
1. Navigate to "Reports"
2. Select report type:
   - Daily Access Log
   - Monthly Summary
   - Homeowner Activity
3. Select date range
4. Click "Generate"
5. Export as PDF or Excel

### For Security Personnel

#### Check-In Process
1. Open scanner interface
2. Click "Check-In Mode"
3. Scan homeowner's QR code
4. Verify displayed information
5. Click "Approve Entry"
6. Gate opens automatically

#### Check-Out Process
1. Open scanner interface
2. Click "Check-Out Mode"
3. Scan homeowner's QR code
4. System shows check-in time
5. Click "Approve Exit"
6. Gate opens automatically

#### Handle Visitor
1. Scan visitor's temporary QR code
2. Verify visitor details
3. Confirm sponsoring homeowner
4. Click "Approve Visitor Entry"
5. Log any additional notes

### For Homeowners

#### Download QR Code
1. Login to homeowner portal
2. Navigate to "My QR Code"
3. Click "Download QR Code"
4. Save to phone or print

#### Register Visitor
1. Login to portal
2. Click "Register Visitor"
3. Enter visitor details:
   - Name
   - Contact number
   - Visit date and time
4. Click "Generate Visitor QR Code"
5. QR code sent to visitor via SMS/Email

#### Update Profile
1. Login to portal
2. Click "My Profile"
3. Update information
4. Click "Save Changes"

#### Register Vehicle
1. Login to portal
2. Navigate to "My Vehicles"
3. Click "Add Vehicle"
4. Enter vehicle details:
   - Plate number
   - Make and model
   - Color
5. Click "Register Vehicle"

## üîß Troubleshooting

### QR Code Won't Scan

**Problem:** QR code not being recognized

**Solutions:**
1. Ensure good lighting
2. Hold QR code steady
3. Clean camera lens
4. Check if QR code is expired
5. Verify QR code is not revoked
6. Try manual entry using homeowner ID

### Homeowner Can't Login

**Problem:** Login credentials not working

**Solutions:**
1. Verify username/email is correct
2. Use "Forgot Password" feature
3. Contact administrator to reset password
4. Check if account is active

### No Check-In Record Found

**Problem:** Check-out attempted but no check-in exists

**Solutions:**
1. Verify homeowner identity
2. Manually create check-in record
3. Add notes explaining situation
4. Proceed with check-out

### System Running Slow

**Problem:** Interface is laggy or unresponsive

**Solutions:**
1. Clear browser cache
2. Check internet connection
3. Restart browser
4. Contact IT support if persists

## üìù Frequently Asked Questions (FAQ)

### General

**Q: How long is a QR code valid?**  
A: Standard QR codes are valid for 1 year from generation date.

**Q: Can I have multiple QR codes?**  
A: No, each homeowner has one active QR code at a time.

**Q: What if I lose my QR code?**  
A: Login to the portal and request a new QR code. The old one will be automatically revoked.

### For Homeowners

**Q: Can family members use my QR code?**  
A: Yes, household members can use the homeowner's QR code, but they should be registered in the system.

**Q: How do I register a visitor?**  
A: Login to the portal, go to "Register Visitor," fill in details, and the system will generate a temporary QR code.

**Q: Can I use my QR code for multiple vehicles?**  
A: Yes, but all vehicles should be registered in your profile.

### For Security

**Q: What if the scanner is not working?**  
A: Use manual entry mode with the homeowner ID or contact IT support immediately.

**Q: How do I handle expired QR codes?**  
A: Politely inform the homeowner and direct them to renew via the portal or contact admin.

**Q: What if someone doesn't have their QR code?**  
A: Verify identity using valid ID, use manual entry, and log the incident.
